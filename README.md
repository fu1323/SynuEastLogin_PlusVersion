某师范大学东区  宿舍网络认证工具
对应配置:ikuai软路由系统、有自己独立域名做ddns（阿里云）
、需要portal才能上网的网络
程序会自己去ikuai后台重制wan口的ip，并获取得到的ip地址。
之后带着地址和帐户信息请求portal认证
最后向阿里云提交更新解析

携带login参数：使用java -jar xxx.jar login运行时会直接登陆，然后退出程序
若不携带参数 直接运行:程序会检测网络状态 判断是否为portal失效状态，失效会执行登陆，未失效会结束程序，
三分钟之后重复

程序初次运行会生成json配置文件，要求填写阿里云key portal信息 ikuai登陆信息（需要抓包，post请求的密码加密值是固定的）
