某师范大学东区  宿舍网络认证工具
对应配置:ikuai软路由系统、有自己独立域名做ddns（阿里云）
、需要portal才能上网的网络
程序会自己去ikuai后台重制wan口的ip，并获取得到的ip地址。
之后带着地址和帐户信息请求portal认证
最后ddns向阿里云提交更新解析（配置文件可关闭）

携带login参数：使用java -jar xxx.jar login运行时会直接登陆，然后退出程序
若不携带参数 直接运行:程序会检测网络状态 判断是否为portal失效状态，失效会执行登陆，未失效会结束 等待下一次调度，
指定时间过后之后重复

程序初次运行会生成json配置文件，要求填写阿里云key portal信息 ikuai登陆信息（需要抓包，post请求的密码加密值是固定的）

jvm会一直运行，定时检查（配置文件可自由指定分钟数)，建议托管给systemd设置开机自启，挂掉拉起
可自由修改  程序接口针对爱快软路由和某大学东区portal系统定制，可灵活修改

更新：
加了个tasmota智能开关控制功能，比如控制显示器 当电脑开机的时候打开，关闭的时候关掉，是通过ping ip判断，由于程序被托管给systemd一直在运行，故顺手把自己用的这个功能加进来了  ddns和tasmota 不需要的话可以在配置文件部分填false 留空不影响ikuai和portal认证的联动部分。
程序运行可携带tasmota参数 只运行这部份逻辑

提示:okhttp会自动url编码，所以配置文件指定portal信息的时候一定要吧url编码转回来（不会出现%）否则okhttp会对内容再进行一次url编码，导致认证失败 服务器报错303
